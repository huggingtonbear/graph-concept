Angular

What is it?

Angular is an open-source JavaScript framework, developed by Google Engineers, built to create dynamic web applications using HTML, JavaScript and CSS. Much like Ruby on Rails, the promise of Angular is to bring simplicity back to the development of complex web applications.

The modern web is far more complex that it was when HTML, CSS, and JavaScript were first developed. Now a single web page can work like an application, letting users enter and receive all kinds of information. Increasingly, a lot of this can be done on the client side - the browser does all the work without ever having to contact a database or a separate application housed on the server. Angular is a sophisticated tool for writing this sort of high-level front-end service.

Who is it for?

The primary audience for Angular is web developers and software engineers who use Angular to create interactive web applications and user-interfaces. They will generally have a thorough grounding in web application development and JavaScript for front-end development.

Customer Benefits

  • Quick and easy to develop applications
  • Easier to test and maintain
  • Build sites that behave like native apps
